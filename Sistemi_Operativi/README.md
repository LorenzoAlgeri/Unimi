## Descrizione

Il corso di Sistemi Operativi è un corso che esplora il funzionamento dei sistemi operativi nel loro insieme, con sezioni dettagliate su argomenti come memoria virtuale, file system, deadlock o schedulazione

## Struttura materiali

- `Lezioni_Lab`: Contiene i materiali del laboratorio del corso, in particolare:
    - `Android`: Contiene le lezioni relative al sistema Android
    - `Introduzione`: Contiene le slide introduttive del laboratorio
    - `Linux`: Contiene le lezioni relative al sistema Linux
    - `Soluzioni_Linux`: Contiene le soluzioni degli esercizi trattati nella parte relativa a Linux
    - `Windows`: Contiene le lezioni relative al sistema Windows
    - `ncfmp.tgz`: Esercizio su comandi terminale Linux
- `Lezioni_Teoria`: Contiene le slide complete del corso del professor Piuri per l'anno 2024/2025
- `Libri`: Contiene i libri del corso
- `Riassunti`: Contiene i riassunti del corso e `TemiEsame.pdf` ovvero la lista delle domande effettuate all'esame
- `argomenti.jpg`: Immagine con la divisione degli argomenti del corso per parte A e parte B
