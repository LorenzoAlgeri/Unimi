## Descrizione

Il corso fornisce nozioni di base di Economia (aziende e gestione aziendale), Etica legata all'informatica, conseguenze sociali e legali dell'informatica nella società, utili per lavorare ed approcciarsi al mondo del lavoro in ambito informatico
